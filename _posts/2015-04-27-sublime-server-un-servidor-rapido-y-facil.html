Cuando desarrollamos ideas rápidas o demos es útil tener un servidor local corriendo. Con el paquete Sublime Server puedes hacer exactamente eso. El paquete te permite iniciar y detener un sencillo servidor de archivos estáticos en el equipo - perfecto para trabajar con HTML, CSS y JavaScript en localhost.

Como bono adicional, el paquete es compatible con Markdown Rendering, por lo que aquellos que utilizan Sublime Text para escribir pueden rápidamente su markdown.

Obtenlo en el Package Control hoy!
